## 背景

DP 状态设计三连（序列/区间/集合）之后补**树形 DP**：状态挂在**树的节点**上、子树天然是子问题、**后序遍历**天然是拓扑序。经典题打家劫舍 III：二叉树上小偷不能偷相邻（父子）节点，求最大金额——每个节点两个状态 `选 = 值 + Σ孩子的不选`、`不选 = Σ max(孩子两态)`，孩子算完父亲才能算。

## 目标

动态规划第 9 页「树形 DP」，接入播放器，**纯复用 MatrixView（第 13 消费者，零改动）**：

1. **节点状态表**：5 行（节点，rowLabels 带树位置 `根 4 / L 1 / R 5 / LL 3 / LR 6`）× 2 列 `[选, 不选]`；**填表按后序跳跃**（行序即树、填序即后序，跳跃本身就是教学点）；内部节点分两步：sel 步 `sources`=孩子的「不选」格、not 步 `sources`=孩子四格（取 max）。
2. 固定完全二叉树 `[4,1,5,3,6]`（层序，Python 已核验）：后序 3→4→1→2→0；叶子 (3,0)/(6,0)/(5,0)；节点 1 选=1/不选=9；根 选=13/不选=14 → **答案 max=14 = 暴力枚举不相邻子集**。**10 步** = init + leaf×3 + (sel+not)×2 + best + done。
3. 正文：树形 DP 三要素（状态挂节点/子树即子问题/后序即拓扑）→ 两态设计与转移 → 没有上司的舞会同款 → 树上背包/换根 DP 点到；链接 TSP/石子合并（状态设计四连收官）。
