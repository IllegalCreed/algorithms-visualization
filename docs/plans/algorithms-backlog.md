## 背景

到 2026-06-29，站点内容已覆盖**数据结构 15 个**（M3 基础 8 + M4 扩充 7：Trie/并查集/LRU/跳表/线段树/B+树/布隆）+ **排序算法 8 个**。但「算法」侧除排序外几乎空白——还有一大批**算法范式/领域**没有覆盖。

## 核心策略（2026-06-29 与 Owner 敲定）

1. **先把后续要补的算法大类全部记录在本池**（本文档），分三阶段排好优先级，避免以后重新盘点。
2. **按「算法范式/领域」新增顶层菜单大类**——和现有「数据结构」「经典排序算法」并列；每个大类下挂多个算法页。
3. **每个大类首发**时：①顺带建**该类共用的可视化基础**（图画布 / DP 表格 / 决策树……，类似当年 C-006 先建算法播放器框架）；②新增**顶层菜单分类**接线；③接入第一个代表算法。之后同类每个算法 = 一个新页。
4. 一项一项走完整流程：设计要点 → 四文档 → TDD（L3→L4→L5）→ 全门禁 → 三索引回写 → 两提交 → 双轨部署。
5. 候选项默认 `candidate`；启动某项再分配 Change ID（顺延全局计数；**开工前先 `grep -rn "C-202..." docs/plans/index.md` 确认号没被别的会话占**）、建四文档、行入 `docs/plans/index.md`。

## 各大类代表算法清单（★ = 建类首发项）

### 图算法 (G) —— 阶段一

> **可视化基础**：扩展现有 `src/components/structures/GraphViz.vue` + `useGraph.ts`（C-022 图数据结构页已实现无向图 + BFS/DFS 基础）为**带权 / 有向图 + 算法走查点亮**（访问中 / 已确定 / 正在松弛的边 / 距离表）。固定图结构（仿数据结构页固定结构的可测做法）。

- **G1 ★ Dijkstra 单源最短路**（非负权）：优先取当前最近点 → 松弛邻边 → 更新距离表。建类首发。
- **G2 BFS / DFS 遍历（算法视角）**：C-022 已讲基础版——本类可**直接链接复用**或加强（无权最短路 = BFS 层），优先级低、可跳过。
- **G3 Bellman-Ford**：可处理负权 + 判负环（V-1 轮松弛）。
- **G4 Floyd-Warshall**：多源最短路（三重循环 DP 矩阵，呼应 DP 类）。
- **G5 拓扑排序**：Kahn 入度法 / DFS 逆后序（DAG）。
- **G6 最小生成树 Kruskal**：排序边 + **并查集**判环（呼应已有并查集页）。
- **G7 最小生成树 Prim**：从点集生长、每次取最小横切边。
- **G8 连通分量 / 强连通分量**：并查集 / Tarjan / Kosaraju。

### 动态规划 (DP) —— 阶段一

> **可视化基础**：新建 **DP 表格**组件——网格逐格填值 + 高亮「当前格」与它「依赖的来源格」(转移箭头) + 终态回溯出最优路径。这是全新范式，首发项要单独设计这套（最值得一次性打磨好）。

- **DP1 ★ 0/1 背包**：二维表（物品 × 容量），逐格 max(不选, 选)。建类首发。
- **DP2 最长公共子序列 LCS**：二维表 + 回溯路径还原序列。
- **DP3 最长递增子序列 LIS**：一维 DP + 二分优化对照。
- **DP4 编辑距离 Levenshtein**：二维表，插/删/替三来源取 min。
- **DP5 完全背包 / 多重背包**：与 0/1 对照（循环方向之差）。
- **DP6 区间 DP**：石子合并 / 矩阵链乘（按区间长度递推）。
- **DP7 入门引子**：爬楼梯 / 斐波那契——记忆化递归 vs 递推对照（可作首发备选，更轻）。

### 回溯与搜索 (BT) —— 阶段一

> **可视化基础**：新建**递归决策树**（节点逐步展开 + 剪枝/回退标记）或**棋盘/网格走查**（放置 + 冲突高亮）。

- **BT1 ★ N 皇后**：棋盘逐行放置 + 列/对角冲突剪枝 + 回退。建类首发。
- **BT2 全排列**：决策树展开 + 已用标记。
- **BT3 组合 / 子集生成**：选 / 不选的二叉决策。
- **BT4 数独求解**：格子候选 + 回溯。
- **BT5 迷宫 / 岛屿 DFS**：网格回溯（呼应图遍历）。
- **BT6 括号生成 / 电话号码字母组合**：经典回溯小品。

### 字符串算法 (STR) —— 阶段二

> **可视化基础**：复用现成**算法播放器**（多语言代码 + 行高亮 + 单步 + 变量面板）——主串 / 模式串两排格子 + 双指针 + next 数组。已有 Trie 可呼应。

- **STR1 ★ KMP**：失配函数 next 构建 + 主串模式串双指针不回溯。建类首发。
- **STR2 Rabin-Karp**：滚动哈希匹配。
- **STR3 Manacher**：最长回文子串（中心扩展 + 半径数组）。
- **STR4 Z 函数 / 扩展 KMP**。
- **STR5 AC 自动机**：Trie + fail 指针多模式匹配（呼应已有 Trie）。

### 查找算法 (SR) —— 阶段二

> **可视化基础**：复用排序的柱状条 + 指针（区间 [lo, hi] 收缩高亮）。

- **SR1 ★ 二分查找及变体**：有序数组区间收缩；含找左/右边界、旋转数组。建类首发。
- **SR2 跳跃查找 Jump Search**：分块跨步 + 块内线性。
- **SR3 插值查找**：按值估计位置（均匀分布）。
- **SR4 三分查找**：单峰函数求极值。

### 数学与数论 (NT) —— 阶段三

> **可视化基础**：步骤面板 / 数值逐步变化（多可复用播放器的变量面板）。

- **NT1 ★ 欧几里得 GCD + 扩展欧几里得**：辗转相除可视化。建类首发。
- **NT2 埃氏筛 / 线性筛**：素数筛除标记动画。
- **NT3 快速幂**（含矩阵快速幂）：指数二进制拆分。
- **NT4 质因数分解**。
- **NT5 模运算 / 逆元**。

### 算法思想专题 (PA) —— 阶段三

> **可视化基础**：复用柱状条 / 序列。把零散「技巧」合成一类讲**思想**，单个内容少、合并更合适。

- **PA1 ★ 双指针 + 滑动窗口**：数组上两指针 / 窗口伸缩。建类首发。
- **PA2 前缀和 / 差分**：预处理 + 区间查询（呼应线段树）。
- **PA3 贪心**：活动选择 / 区间调度 / 找零 / 霍夫曼。
- **PA4 分治**：最近点对 / 快速幂思想 / 归并思想回顾。
- **PA5 二分答案**：把「求解」转「判定 + 二分」。
