## 背景

C-079 欧几里得页尾预告：「扩展版还能一并算出让 `ax + by = gcd(a,b)` 成立的整数 x, y（Bézout 系数）——那是这条线自然的下一站」；C-080 快速幂页点出模逆元是 RSA 一步。本页补上这块拼图。

**扩展欧几里得**：在辗转相除求 gcd 的同时，解出 **Bézout 系数** `x, y` 使 `ax + by = gcd(a,b)`。**下行**照常做除法链（`a = q·b + r`）；到基例 `b = 0` 时 `gcd = a`，显然 `a·1 + 0·0 = a`，即 `(x,y) = (1,0)`；**回代**自底向上：若下层给出 `b·x' + r·y' = g`，代入 `r = a − q·b` 整理得 `a·y' + b·(x' − q·y') = g`，故上一层 `(x, y) = (y', x' − q·y')`。有了 Bézout 系数，`gcd(a,m)=1` 时 `x mod m` 就是 **a 的模逆元**——RSA 解密指数、除法取模的基石。

## 目标

数学与数论第 5 页「扩展欧几里得」，接入播放器，**纯复用 MatrixView（第 8 消费者，零改动）**：

1. **系数回代表**：5 列 `[a, b, q, x, y]` × 4 行。下行阶段逐行填 `(a,b,q)`；基例行填 `(6,0,−,1,0)`；回代阶段自底向上填 `x, y`（`sources` 高亮引用的下一行 x'、y'，caption 给公式）。
2. 固定 `30, 18`（与 GCD 页同款）：除法链 `30=1·18+12 / 18=1·12+6 / 12=2·6+0`；回代 `(0,1)→(1,−1)→(−1,2)`；**gcd=6、x=−1、y=2**（`30·(−1)+18·2=6`），每层回代均验证 `a·x+b·y=6`。
3. 正文：Bézout 系数 → **模逆元**（`3⁻¹ mod 7 = 5`）→ RSA/同余方程应用。
