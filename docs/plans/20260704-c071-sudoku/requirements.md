## 背景

回溯与搜索大类已有 7 页：N 皇后（棋盘约束）、子集/排列/组合总和（决策树）、迷宫/岛屿/单词搜索（网格）。其中「棋盘约束」只有 N 皇后一例（在列上放皇后、判对角冲突）。**数独**是棋盘约束回溯的另一经典形态，也是「**约束满足 + 回溯**」最有名的例子：每个空格试填 1..n，要求所在**行、列、宫**都不重复；填不下去就**撤销、回退**换一个数。

N 皇后是「放置型」回溯（每列放一个皇后），数独是「**填数型**」回溯（每个空格填一个数、受三重约束），两者互补，把「棋盘约束」这一状态空间讲全。

## 目标

在回溯与搜索大类新增第 8 页「数独」，接入算法播放器（`AlgorithmPlayer`）：

1. **新建 SudokuView 数独轨**（第 14 条播放器轨，additive 可插拔）：`n×n` 数字盘 + `√n × √n` 宫线；初始给定数字加粗、填入数字常规、空格；当前格琥珀环、试填数字冲突标红、填入标绿。
2. **试填 + 约束检查 + 回溯逐步重走**：`reject`（试某数字，与行/列/宫冲突 → 换下一个）→ `place`（试某数字不冲突 → 填入）→ `backtrack`（一个空格 1..n 都填不了 → 撤销上一个填入、回退）→ `done`（全部填满）。
3. 固定输入：4×4 迷你数独（2×2 宫），5 个空格，唯一解；过程含 **2 次真回溯**——先在某空格填了一个「当下合法但全局错误」的数，走到死路后撤销、改填正确的数。约 **22 步**。

## 验收标准

- `/docs/sudoku` 新页：介绍正文（讲清行/列/宫三重约束 + 试填-冲突-回退）+ 播放器同屏，右侧四语言代码随步高亮。
- 动画：4×4 盘（宫线可见）、给定数字加粗、当前格琥珀、冲突红、填入绿；过程有明显的「填错 → 死路 → 撤销回退 → 改填」。
- 菜单 + 首页「回溯与搜索」新增第 8 项，新图标 `sudoku.svg`。
- 全门禁绿（format/lint/type-check/单测覆盖率/e2e）；真机自检确认三重约束、2 次回溯、终盘正确；既有算法零回归。

## 非目标

- 不做 9×9（步数太多；4×4 已足够展示约束回溯且含真回退）。
- 不做位运算 / Dancing Links 等优化——本页聚焦朴素约束回溯。
- 不改 AlgorithmPlayer 既有轨；SudokuView 为新增 additive 轨。
