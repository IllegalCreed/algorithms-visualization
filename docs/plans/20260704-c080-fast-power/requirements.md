## 背景

数学与数论大类已有埃氏筛 / 线性筛（素数）+ 欧几里得（GCD）三页。本页补另一块数论基石——**快速幂（二进制取幂 / binary exponentiation）**。

**快速幂**：求 `aⁿ`，朴素做法乘 n 次是 O(n)。快速幂把指数 n 拆成**二进制** `n = Σ 2ᵏ`（对每个为 1 的位），于是 `aⁿ = ∏ a^(2ᵏ)`。底数从 `a` 开始**反复平方**得到 `a¹ → a² → a⁴ → a⁸ → …`（每步平方一次），扫描 n 的二进制位：位为 1 就把当前的平方**乘进结果**，位为 0 就跳过。这样只需 **O(log n)** 次乘法。它是**模幂运算**（`aⁿ mod m`，RSA 加解密、Diffie-Hellman 的核心）与矩阵快速幂（线性递推加速）的基础。

## 目标

在「数学与数论」大类新增第 4 页「快速幂」，接入算法播放器（`AlgorithmPlayer`），**新建第 18 条 PowerView 幂块轨**：

1. **幂块行**：把底数的各个平方 `a¹, a², a⁴, a⁸` 画成一行卡片，每张卡显示指数（`a^(2ᵏ)`）、值、以及对应的二进制位；位为 1 的卡**选中**（绿），位为 0 的卡**跳过**（灰），当前处理的卡**琥珀**高亮。
2. **逐位累乘**：`init`（展示 n 及其二进制）→ 逐位 `mul`（位=1：底数平方出新卡、选中、结果乘入）或 `skip`（位=0：出新卡但不乘）→ `done`（结果 = 选中幂块之积 = aⁿ）。
3. 固定 `a=3, n=13`（`13 = 1101₂`）：幂块 `3¹=3, 3²=9, 3⁴=81, 3⁸=6561`，选中 `3¹·3⁴·3⁸`（指数和 1+4+8=13）= **3×81×6561 = 1594323 = 3¹³**。
