## 背景

现有七大顶层分类（数据结构 / 排序 / 图算法 / 动态规划 / 回溯与搜索 / 字符串 / 数学与数论）。M6/M7 里程碑「算法分类扩充」还预留了**计算几何**方向。本页开辟第 8 个顶层大类，首发最经典、最直观的几何算法——**凸包（Convex Hull）**。

**凸包**：把平面上一堆点用**最紧的凸多边形**套住（像橡皮筋绷住外围的钉子），内部的点被排除。**Andrew 单调链**：先把点按 `(x, y)` 排序；从左到右扫描构**下凸壳**、从右到左扫描构**上凸壳**。维护一个栈，每加入一个点，就检查栈顶两点与新点的**转向**——用**叉积** `cross(O,A,B) = (A−O)×(B−O)` 判断：若 ≤ 0（**非左转**、即右转或共线），说明栈顶点是凹的，**弹栈**；直到左转再把新点压入。下凸壳 + 上凸壳拼起来就是完整凸包，排序 O(n log n) 主导、扫描 O(n)。**叉积的正负 = 转向的左右**，是计算几何最核心的原语。

## 目标

新建顶层大类「计算几何」，首页 1 页「凸包」，接入算法播放器（`AlgorithmPlayer`），**新建第 19 条 HullView 点平面轨**：

1. **点平面**：把点画在平面上（y 轴上翻贴合数学直觉），当前凸壳用**折线**连接，当前处理的点**琥珀**高亮，本步因右转被**弹出**的点标红，终局凸包连成**闭合多边形**。
2. **单调链**：`init`（散点）→ 逐点 `lower`（下凸壳：加点、右转弹栈、压入）→ 逐点 `upper`（上凸壳）→ `done`（下 + 上拼成凸包）。
3. 固定 7 点 `(0,3),(2,0),(2,5),(3,3),(4,0),(4,5),(6,3)`：凸包 6 点 `(0,3),(2,0),(4,0),(6,3),(4,5),(2,5)`，内部点 `(3,3)` 被排除；下凸壳、上凸壳各发生几次弹栈。

## 非目标

- 不做 Graham 扫描 / 礼品包裹（Jarvis）/ QuickHull 等其它凸包算法——本页专注 Andrew 单调链，为计算几何大类铺 HullView 轨。
- 不做三维凸包 / 动态凸包——固定平面整点集。
- 不改 AlgorithmPlayer 既有轨；仅 additive 加 `Step.hull?` + 一行 `v-if`（其它算法不设 → HullView 不渲染，零回归）。
