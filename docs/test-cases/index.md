## 使用说明

本文件保存测试用例事实字段（Case ID、owner plan、自动化路径、状态、最后验证）。

本项目测试分层裁剪为 L3（前端单元）/ L4（前端组件）/ L5（端到端），不含 L1/L2（无后端）。
