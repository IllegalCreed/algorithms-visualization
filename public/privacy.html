隐私与统计说明

最后更新：2026-07-10

当前统计接入仍在准备阶段。在 Umami Cloud 托管区域与实际数据保留期限完成确认前不会启用生产统计。

用途

统计只用于判断哪些学习页面和交互真正有帮助，以及不同公开发布渠道是否值得继续投入。统计结果不会用于广告画像、跨站追踪或识别个人。

启用后会处理的数据

  • 去除查询参数和片段后的页面路径、页面标题及访问时间。
  • 经过白名单校验的 UTM 来源、媒介、campaign 与 content。
  • 外部来源只保留 hostname；不会发送 referrer 的路径、查询参数或会话标识。
  • 浏览器、设备类型、语言、屏幕尺寸与匿名化的粗粒度地区信息。
  • 搜索、播放、应用输入、完成测验和分享等事件名称及有限计数。

明确不会收集的数据

本站不发送搜索词原文、算法输入值、测验题目或选项、姓名、邮箱、账号、自定义 User ID、设备指纹或完整 referrer。统计脚本不使用统计 Cookie，也不启用广告、回放、热图或跨站识别功能。

与任何网络服务相同，请求 Umami Cloud 时服务端会看到源 IP。Umami 官方说明会结合 IP、User-Agent 与 website ID 派生匿名会话，并用于地区统计；本站不会把 IP 写入事件属性或自定义身份字段。

浏览器存储

为在一次浏览会话内区分首触与当前渠道，本站只在 sessionStorage 中保存四个经过校验的 UTM 字段或来源 hostname。关闭该标签页后浏览器会清除这些会话数据。

服务提供方与退出方式

计划使用 Umami Cloud 的 EU 区域。tracker 设置为尊重浏览器的 Do Not Track，并只允许本站自有域和 GitHub Pages 镜像上报。你可以启用 Do Not Track，或在浏览器中阻止 cloud.umami.is；站点内容与全部算法交互仍可正常使用。

保留、导出与删除

项目目标是原始统计数据保留不超过 180 天。Umami Cloud Hobby 的实际期限和删除能力将在账号内核实后更新本页；若无法执行该目标，必须先明确调整方案，不能静默延长。Cloud 官方提供数据导出和账号删除能力。

联系

对本说明或数据处理有疑问，可在 GitHub Issues 提出。
